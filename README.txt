*Project description:
	-Name: chess with AI
	-In this project I will created chess game that enables two players to play against each 	
	Other and allows player to play against a bot using AI

*How to run:
	-the project consists of 4 files:
		1- main_chess: this file is responsible for the GUI , this is the file that the user should run
		2- chessengine: this file is responsible for the chess engine
		3- chessAI: this file is responsible for the AI 
		4- Images: this file cointains the chess peices images

*Libraries:
	the user needs to have pygames installed, this could be done by copying 'py -m pip install -U pygame --user' into the command line
